# Quickstart Guide

Get up and running with CortexEDR in under five minutes.

## 1. Connect a Repository

Navigate to the dashboard and connect your GitHub account via OAuth. Click **New Scan** and select your desired repository. 

## 2. Initialize Scan

Cortex begins cloning your repository and analyzing its abstract syntax tree (AST). You'll see real-time agent thoughts tracking vulnerabilities and potential architecture improvements in the **Live Event Feed**.

## 3. Review Manifest

Once complete, access the detailed Audit Manifest to explore your security score, findings, and technical analysis.
