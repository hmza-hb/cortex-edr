# Frequently Asked Questions

**Q: Does Cortex store my repository code?**
A: No. Code is cloned temporarily into our secure vault to build the AST and generate the Audit Manifest. Once the scan completes, all local assets are securely wiped.

**Q: Which languages are supported?**
A: TypeScript, JavaScript, Python, Go, Rust, and Java components are fully mapped and deeply analyzed.

**Q: How do we upgrade to the Enterprise tier?**
A: Use the **Upgrade Plan** button in your dashboard or contact our sales team through the Support portal.
