# Quickstart Guide

Get up and running with CortexEDR in less than 60 seconds. Our platform is designed to be frictionless, agentic, and powerful.

## 1. Connect Your Network
The first step is granting Cortex access to your source code. We support both public and private repositories via GitHub OAuth.

1.  Navigate to the **Repositories** tab in your [Dashboard](/dashboard/repositories).
2.  Click **Connect GitHub Account**.
3.  Select the repositories you want to protect.

> [!TIP]
> You can also manually import any repository by providing its public URL in the **New Scan** interface.

## 2. Trigger Neural Intelligence
Once your repository is connected, you can launch a deep-pass security audit.

1.  Go to the **New Scan** page.
2.  Select your repository and choose your **Intelligence Tier**.
3.  Click **Initiate Protocol**.

| Tier | Reasoning Engine | Depth |
| :--- | :--- | :--- |
| **Vibe Coder** | Liquid LFM | Fast / Surface |
| **Developer** | DeepSeek R1 | Deep / Logic |
| **Teams** | Claude 3.5 Sonnet | Enterprise / Full |

## 3. Analyze the Manifest
Cortex will deploy its 7-agent pipeline. You can watch the real-time feedback in the **Agent Live Feed**. 

Once complete, you will receive a **Security Manifest** containing:
- **Vulnerability Heatmap**: Priority-sorted CWE findings.
- **Architectural Debt**: High-level mapping of structural risks.
- **Automated Fix Solutions**: One-click PRs to resolve identified issues.
