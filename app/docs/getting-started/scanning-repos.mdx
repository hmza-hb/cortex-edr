# Scanning Repositories

CortexEDR utilizes advanced, multi-agent AI architecture to scan entire repositories dynamically.

## Supported Platforms

Currently, Cortex natively supports GitHub integration. Bitbucket and GitLab are in development.

## The Scanning Process

1. **Cloning**: Shallow clones securely pull down your main branch.
2. **Analysis**: Static pattern checks combined with deep AI semantic understanding.
3. **Synthesis**: Correlating scattered symptoms into distinct root vulnerabilities.
